📓 Note Taker

Your personal AI-powered note assistant

💡 What is Note Taker?

Note Taker is a private, AI-powered note-taking app that runs entirely in your browser as a Progressive Web App (PWA). You can dictate or type notes, then ask questions about them in plain English — the AI reads your notes and answers from them.

All notes are stored locally on your device using IndexedDB. Nothing is sent to any server except the text of your notes when you make a query or transcribe a recording.

🚀 Getting Started

1
Install the app

In Chrome, tap the menu → Install app to add it to your home screen for full-screen, app-like use.

2
Enter API keys

Tap Unlock with the shared access key, or enter your own Anthropic and OpenAI keys in Settings.

3
Choose a category

Each category has its own independent notes and context. Create one on first use — for example Work, Personal, or Health.

4
Start noting

Dictate or type a note and tap Save, or ask a question about your existing notes.

🎛️ The Toolbar Buttons

🎙️ Mic — Dictate 📓 Save Note 🔍 Query ✏️ Edit ✕ Clear text
  • Mic (dictate): Tap once to start recording. Speak your note. Recording stops automatically after 2 seconds of silence, or tap again to stop manually. The audio is transcribed and placed in the text box.
  • Save Note: Saves whatever is in the text box as a note. By default it files under today, but if your text includes a date instruction (e.g. "add this to yesterday" or "file under April 5") it will save to that date instead. Multiple saves in one day accumulate.
  • Query: Asks the AI a question using the text in the box. The AI reads all your relevant notes and searches the web before answering. The question and answer appear directly below the text box.
  • Edit: Opens the note editor. Speak or type a date (e.g. yesterday, April 3) to load that day's notes and edit them directly.
  • Clear (✕): Appears inside the text box whenever there is text. Tap it to clear the input in one tap.

🎙️ Dictating Notes

Tap the orange microphone button to start. The button pulses red while recording. Speak naturally — the app uses OpenAI Whisper for accurate transcription.

Recording stops automatically after 2 seconds of silence once you have started speaking, or you can tap the button again to stop immediately.

The transcribed text is placed in the main text box. You can edit it before saving.

Tip: You can dictate a note then tap Save, or dictate a question then tap Query — it works either way.
Note: Transcription requires your OpenAI key and an internet connection. The app will ask for microphone permission on first use.

🔍 Querying Your Notes

Type or dictate a question, then tap the Query button. The AI (Claude) searches the web and reads your notes, then answers using both sources. Your question stays visible in the text box and the answer appears directly below it.

Example questions:

  • "What did I eat last week?"
  • "Summarise my notes from this month."
  • "Did I mention anything about the dentist?"
  • "What tasks have I noted that are still outstanding?"
  • "What is the weather forecast for London?"
Tip: For questions answered by your notes, Claude cites the timestamps. For questions not in your notes, it falls back to web search results and cites the source URLs.

✏️ Editing Notes

Tap the pencil (Edit) button to enter edit mode.

  1. Type or speak a date — for example "yesterday", "Monday", or "March 15".
  2. Tap the arrow to load the notes for that day.
  3. Edit the text directly in the editor.
  4. Tap Save to write the changes back.

Edits are marked with an [Edited: …] timestamp appended to the note.

📁 Categories

Notes are organised into categories — each with its own completely independent set of notes and AI context. This lets you keep different areas of your life separate, for example:

  • Work — meetings, tasks, decisions
  • Personal — diary, ideas, reminders
  • Health — symptoms, appointments, medication
  • Project X — anything specific to a project

On every reload you are shown the category picker. Categories are listed alphabetically and the list scrolls, so it works equally well with a handful of categories or dozens. Tap a category to open it, or type a new name and tap Start to create one. To switch category mid-session, tap the folder button in the top-right corner — this returns you to the category picker instantly.

To delete a category and all its notes, tap the 🗑 button next to it on the picker screen. You will be asked to confirm before anything is deleted.

When you query your notes, the AI only sees notes from the currently selected category, keeping contexts clean and separate.

Note: Category names and notes are stored locally in your browser. Clearing browser data will remove them.

💾 Backup & Restore

The category selection screen has two buttons at the bottom for backing up and restoring all your notes across every category:

  • ⇓ Export All — downloads a single .json file containing every category and all their notes. Save this to Google Drive or email it to yourself regularly.
  • ⇑ Import — pick a previously exported .json file to restore. Any categories in the file are added if missing, and existing notes for matching dates are overwritten with the imported content.
Important: Notes are stored in your browser's IndexedDB. Clearing browser site data will erase them permanently. Export regularly to avoid data loss.
Tip: After a restore, return to the category screen to see all imported categories listed and ready to use.

🔑 API Keys & Settings

The app needs two API keys to function:

  • Anthropic key — powers the AI query feature (Claude).
  • OpenAI key — powers voice transcription (Whisper).

You can either:

  • Unlock shared keys — enter the access key provided by the app administrator. This decrypts pre-stored keys so you don't need your own accounts.
  • Enter your own keys — paste your personal API keys from console.anthropic.com and platform.openai.com.

Keys are stored in your browser's local storage. Open Settings at any time via the gear icon in the top-right corner.

Privacy: Your API keys never leave your device except as authorisation headers sent directly to Anthropic and OpenAI.

📲 Installing & Offline Use

Note Taker is a Progressive Web App (PWA). To install it:

  1. Open the site in Chrome on Android.
  2. Tap the three-dot menu → Install app.
  3. The app appears on your home screen and opens full-screen without a browser bar.

Once installed, the app shell loads offline. However, saving notes, querying, and transcription all require an internet connection to reach the AI APIs.
